Apresentação cinematográfica para homenagens — slides sincronizados à música, Ken Burns, vídeos de IA e ornamentos botânicos em SVG.

Ver no GitHub Download ZIP

Recursos

🎬

Slides sincronizados

Cada slide entra e sai no segundo exato da música, controlado por audio.currentTime.

📸

Ken Burns preciso

Zoom lento nas fotos com background-position por slide — rostos nunca cortados.

🎥

Vídeos de IA

Integração com Google Veo 3.1 — micro-vídeos como fundos cinematográficos nos momentos de clímax.

🌿

Ornamentos SVG

Ramos botânicos, molduras, coroa e divisores dourados — todos inline, sem dependências externas.

✍️

Tipografia elegante

Cormorant Garamond para textos e Cinzel para títulos de ato — hierarquia visual clara.

⚙️

Zero dependências

HTML, CSS e JavaScript vanilla. Abre direto no navegador, sem servidor ou npm.

Como Usar

1

Baixar o template

Clone o repositório ou baixe o ZIP. Copie suas fotos para assets/ e a música para musica.mp3 na raiz.

2

Editar o array SLIDES

Abra homenagem.html e edite o array SLIDES com os caminhos das suas fotos, os tempos de cada slide e os textos desejados.

3

Adicionar vídeos de IA (opcional)

Gere micro-vídeos no Google Veo 3.1, salve em revividas/ e referencie com o campo video: nos slides de maior impacto.

4

Abrir no navegador

Clique em homenagem.html — abre direto. Para gerar um MP4, use os scripts montagem/*.ps1 com FFmpeg.
